## 🆘 TROUBLESHOOTING

### **Homepage Not Displaying Correctly?**
- Check browser compatibility (use Chrome 90+)
- Clear browser cache
- Verify all image paths are correct
- Check console for errors (F12)

### **Fonts Not Showing?**
- Check internet connection (Google Fonts requires online)
- Verify font import in CSS
- Try system fallback fonts temporarily

### **Images Too Large?**
- Compress images (use TinyPNG or ImageOptim)
- Reduce image dimensions
- Use WebP format for smaller file size

### **Mobile Menu Not Working?**
- Check JavaScript is enabled
- Verify hamburger menu CSS media query
- Test in different mobile browsers

### **Performance Issues?**
- Optimize and compress all images
- Minimize CSS and JavaScript
- Enable gzip compression on server
- Use CDN for images
